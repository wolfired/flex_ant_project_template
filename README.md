```
  ├─rsl                         #Global runtime Share libaray
  └─src
```








####To SVN User, before the first commit you need to:
1. add the follow ignore list to the root dir:
```
bin
log
ref
config4*.properties
flex_config_*.xml
```
2. edit the **config.properties.base** and set
```
config.is_use_svn=true
```

###Good Luck!!
